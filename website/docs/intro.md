# Introduction

## Document Scope

DataReach is an RSA Professional Services add-on to the SecurID Identity Governance & Lifecycle product. It facilitates the collection of user and access information from multiple endpoints of the same type.

This guide outlines the steps required to install all components of DataReach.

## Intended Audience

This document is for SecurID IG&L administrators responsible for installing and configuring DataReach.
